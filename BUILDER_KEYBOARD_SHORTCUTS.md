# VibeShack Builder — Keyboard Shortcuts Reference

Print this card or keep it nearby while building.

---

## ⌨️ Essential Shortcuts

| Shortcut | Action | Use When |
|----------|--------|----------|
| `Cmd/Ctrl+S` | Save draft | Making important changes |
| `Cmd/Ctrl+Z` | Undo last action | Made a mistake |
| `Cmd/Ctrl+Shift+Z` | Redo last action | Undid too much |
| `Esc` | Close modal / Cancel | In a text editor or dialog |
| `Tab` | Navigate forward | Moving through UI elements |
| `Shift+Tab` | Navigate backward | Going back through UI elements |

## ✏️ Text Editing

| Shortcut | Action |

## 💡 Pro Tips

### Speed Up Your Workflow
1. **Save frequently**: Cmd/Ctrl+S every 5 minutes
2. **Use undo liberally**: You have 20 undo steps
3. **Preview often**: Switch to Preview mode to check quality
4. **Mobile-first**: Start with mobile, work up to desktop

### Muscle Memory
- Memorize the top 5 shortcuts
- Practice for 5 minutes
- After that, they become automatic

### Accessibility
- All buttons have keyboard equivalents
- Tab to navigate UI
- Space to activate buttons
- Arrow keys to adjust values

## Mac vs Windows/Linux

Some shortcuts vary by OS:

| Mac | Windows/Linux | Action |
|-----|---------------|--------|
| `Cmd` | `Ctrl` | Modifier key |
| `Option` | `Alt` | Alternative modifier |
| `Return` | `Enter` | Confirm action |
| `Delete` | `Backspace` | Delete character |

## One-Handed Shortcuts

Use left hand only:
- `Q` - Undo
- `W` - Redo
- `S` - Save
- `P` - Preview
- `R` - Responsive
- `X` - Delete
- `V` - Paste
- `C` - Copy

---

## Customizing Shortcuts

Want to change shortcuts to match your workflow?

1. Go to Settings (gear icon)
2. Click "Keyboard Shortcuts"
3. Click any shortcut to customize
4. Your custom shortcuts sync across sessions
