# VibeShack Builder — Quick Start Guide

**Welcome to the visual page builder for VibeShack Studios.**

## 🎯 Your First 5 Minutes

### 1. Login
Navigate to `/builder` and enter your builder password.

### 2. The Interface
Three main areas:
- **Left Panel**: Manage sections, photos, history, layout
- **Center Canvas**: Live preview of your page
- **Top Bar**: Save, publish, undo/redo, preview mode

### 3. Add a Section
1. Click the **Sections** tab (left panel)
2. Click "Add Section"
3. Choose from templates: Hero, Studio Grid, Differentiators, Testimonials, Gallery
4. Your section appears on the canvas instantly

### 4. Edit Content
1. Click on any element to select it
2. The element editor appears in the left panel
3. Edit text, images, styling
4. Changes save automatically every 30 seconds

### 5. Upload Photos
1. Click the **Photos** tab
2. Click "Upload Photos"
3. Select multiple images at once
4. Photos appear in your gallery instantly

### 6. Save & Publish
- **Save**: Ctrl/Cmd+S (saves to draft)
- **Publish**: Red button in top bar (goes live instantly)

## 🎨 Design System

### Colors
- **Background**: Pure black (#000000)
- **Text**: White (#FFFFFF)
- **Accent**: VibeShack Red (#DC2626)
- **Subtle**: Gray-400 for secondary text

### Typography
- **Headlines**: Bold, clean hierarchy
- **Body**: Regular weight, generous line-height
- **Small Text**: Gray-500 for hints and timestamps

### Spacing
- **Padding**: 4px grid (4, 8, 12, 16, 24, 32px)
- **Gaps**: Generous whitespace between sections
- **Margins**: Let content breathe

### Buttons
- **Primary**: Red background, white text
- **Secondary**: Subtle gray with hover effect
- **States**: Disabled are 50% opacity

### Transitions
- All animations: 200-300ms
- No jarring changes
- Smooth easing

## 🚀 Advanced Features

### Responsive Preview
Click the **Responsive** button to see your page on:
- Desktop (1920×1080)
- Tablet (768×1024)
- Mobile (375×667)

### Version History
Click the **History** tab to:
- View all saved versions
- Revert to any previous version
- See who made changes and when

### Image Editing
1. Select a photo in the **Photos** tab
2. **Filters**: Adjust brightness, contrast, saturation
3. **Crop**: Resize with aspect ratio options (1:1, 16:9, 4:3)
4. Click "Apply" to save

### Photo Reorder
Click the **Reorder** tab to:
- Drag photos to new positions
- Organize by section
- Batch operations

## 💾 Auto-Save & Publishing

### Auto-Save
- Saves every 30 seconds automatically
- Shows "Last saved: 2:45 PM" in top bar
- You can always undo

### Publishing
1. Click the red **Publish** button
2. Confirm the dialog
3. Page goes live instantly
4. Previous versions are archived automatically

## 🆘 Tips & Tricks

### Make Changes Fast
- Use keyboard shortcuts
- Click to select, type to edit
- Drag to reorder

### Preview Like a User
- Click **Preview Mode** to see what visitors see
- No edit tools visible
- Full-screen experience

### Keep It Organized
- Use descriptive section titles
- Group related photos
- Delete unused versions from history

### Mobile-First Thinking
- Check **Responsive** view early
- Test on actual devices
- Optimize images (< 500KB each)
