# VibeShack Builder — Video Tutorial Script

A complete script for creating tutorial videos. Each section is ~2-3 minutes.

---

## 📹 Overview

- **Total Series**: 5 videos, 10-15 minutes total
- **Format**: Screen capture + voiceover
- **Audience**: First-time builders, non-technical
- **Tone**: Friendly, encouraging, clear

## Production Notes

### Camera & Audio
- **Screen Capture**: OBS or ScreenFlow, 1440p
- **Voiceover**: Quiet room, good microphone
- **Music**: Soft background during intro/outro
- **Pacing**: Speak clearly, pause between sentences

### Editing Tips
1. Cut out pauses and mistakes
2. Add text overlays for key shortcuts
3. Zoom in on UI elements for clarity
4. Use callout arrows to highlight buttons
5. Keep fast cuts, no long static scenes

### Timestamps
- Create chapters for each section
- Link in description to Video 1-5
- Make series playlist

### File Naming
- `Builder_01_Welcome_and_Login.mp4`
- `Builder_02_Creating_Sections.mp4`
- `Builder_03_Managing_Photos.mp4`
- `Builder_04_Responsive_and_Preview.mp4`
- `Builder_05_Save_and_Publish.mp4`

### Thumbnail Design
- Red VibeShack branding
- Large text with shortcut/feature
- Example: "🎯 Add Sections in Seconds"
